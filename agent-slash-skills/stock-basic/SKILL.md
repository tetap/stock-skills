# 基本面分析

用户在本条消息 **`/stock-basic` 或 `$stock-basic` 后面**的文字是股票名称或代码。

## 你要做的

1. 读取并遵循 **stock-fundamental-analysis** skill
2. 拉取公司简介、财报、估值
3. 输出概况、财务趋势、风险点
4. 末尾加免责声明
