# A 股主命令

| 输入 | 行为 |
|------|------|
| `600519` / `贵州茅台` | 查现价 |
| `分析 XX` / `能不能买` / `看几日线` | **全量拉数** + 6 节报告 |

## 分析必做

1. **stock-main** + **analysis-report.md** 工具表（≥18 次调用，基本面 income/balance/cashflow 全拉）
2. 简洁 6 节报告（表格为主）：结论与近期操作、基本面、技术面、资金筹码、事件、风险
3. **禁止**顾问角色、禁止只拉 2 个工具就写报告、禁止踢皮球、禁止必买
