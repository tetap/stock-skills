# A 股主命令

用户在本条 `/stock` 后面输入：查价，或 **分析/能不能买/看几日线** 等。

## 示例

```
/stock 600519
/stock 分析 黔源电力，看几日线能不能介入
```

## 你要做的

1. **stock-main**（查价 → quick-lookup；分析 → 全量拉数）
2. 分析时读 **stock-main/analysis-report.md**，按表 **逐条调用 MCP/CLI**（≥18 次，基本面三表缺一不可）
3. **禁止**只拉 quote + 一条 kline 就写报告；缺数据在报告中标注，不要编造
4. 输出 **6 节简洁报告**（表格为主，少废话），§1 含看几日线、操作倾向、介入/止损价位
5. **禁止**投资顾问角色（buffett/graham 等）、禁止踢皮球 redirect
6. 末尾免责声明
