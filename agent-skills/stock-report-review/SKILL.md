# 报告多重审核

**何时用**：`/stock 分析`、板块选股、市场热点等需要 **出报告** 的场景；**查价不用**。

**完整协议**：[../stock-main/review-protocol.md](../stock-main/review-protocol.md)

---

## 快速流程（个股）

```
R0 拉数(≥20) → R1 初稿(内部) → R2 审计 → R3 魔鬼质疑(≥5问)
→ R4 一致性矩阵 → R5 压力测试 → R6 门禁 → 终稿+§7审核纪要
```

---

## 三轮最低要求

### R2 数据审计

- 每个数字有工具来源
- gaps 不得隐瞒
- P0 缺口 → 补拉后再审

### R3 魔鬼质疑（至少 5 问）

覆盖：基本面 / 技术+quant / 资金 / 情绪 / 交易计划

无法反驳 ≥3 条 → 评级降一档

### R4 一致性

必做表格：158 vs 360 5d/60d、quant vs MA、资金 vs 价格、财报 vs 估值

---

## 置信度

- 初稿 ≤6
- 终稿 ≥7 才可「积极跟踪」类强立场
- ≤4 禁止给买卖区间
