# 快速查价

## Workflow

1. `resolve_symbol --query "{用户输入}"`
2. `get_realtime_quote --secid {secid}`

## 输出格式

```markdown
## {名称} ({代码})

| 最新价 | 涨跌幅 | 涨跌额 | 今开 | 最高 | 最低 | 换手率 |
|--------|--------|--------|------|------|------|--------|
| ...    | ...    | ...    | ...  | ...  | ...  | ...    |

> 数据仅供参考，不构成投资建议。
```

## 注意

- 只回答当前行情，不做深度分析
- 由 **stock-main** 在「仅查价」意图下调用
- 用户要分析/投资建议时，转交 **stock-main**（勿在此展开）
