# 板块分析与选股模板

用于 `/stock 电池板块最近走势，给我几个看好的股票` 等 **板块 + 选股** 请求。

---

## 禁止

- 踢皮球、只列板块名不分析
- 必涨必买、全仓
- 投资顾问流派章节

---

## 必拉工具（≥12 次，再对候选股做轻量扫描）

### 1. 定位板块

| 工具 | 参数 |
|------|------|
| `search_sectors` | query=用户说的板块词（如「电池」） |
| 若多个匹配 | 选最相关 1 个，报告中说明选用哪个 |

### 2. 板块全景

| 工具 | 参数 |
|------|------|
| `get_sector_detail` | board_name, kline, limit 120 |
| `get_sector_detail` | board_name, fund_flow, limit 30 |
| `get_sector_detail` | board_name, members, limit 30 |
| `get_sector_overview` | concept + industry 各 limit 20（看轮动） |

### 3. 新闻与情绪（必做）

| 工具 | 参数 |
|------|------|
| `get_market_news` | flash, limit 30 |
| `get_market_news` | flash, keyword=板块词, limit 15 |
| `get_market_news` | headline, keyword=板块词, limit 10 |
| `get_market_fund_flow` | limit 10 |
| `get_fund_flow_rank` | limit 20 |

从快讯/要闻中提炼 **2~4 条与板块相关的热点**，写进报告「情绪面」。

### 4. 候选股轻量扫描（选 3~5 只）

从 **资金流入靠前 + 涨幅适中 + 非 ST** 的成分股里挑 3~5 只，每只至少：

| 工具 | 说明 |
|------|------|
| `resolve_symbol` + `get_realtime_quote` | 现价与涨跌 |
| `get_valuation_metrics` | PE/PB |
| `get_stock_fund_flow` | 近 5 日主力 |
| `get_news_and_reports` | news limit 5 |
| 可选 | `get_kline` limit 60、`compare_performance` |

**不必**对每只走完整 18 工具个股分析，但要有数据支撑的「为什么选它」。

---

## 报告结构

```markdown
# {板块名} 板块 · 走势与选股

> 数据截止：{时间} · 匹配板块：{实际板块名}（{industry/concept}）

## 1. 结论

**板块状态**：{偏强|中性|偏弱} · **情绪**：{热点驱动|观望|退潮}

| 项目 | 内容 |
|------|------|
| 板块趋势 | 近 20/60 日（来自板块 K 线） |
| 资金 | 板块内主力方向 + 大盘背景 |
| 情绪热点 | 2~3 条快讯/政策要点 |
| 操作倾向 | 可跟踪龙头 / 等回调 / 暂不参与 |

## 2. 板块走势

| 近 5/20 日涨跌 | 涨跌家数 | 领涨 |
| MA 趋势（简述） | |

## 3. 情绪与热点

- 7×24 与板块相关的 2~4 条
- 对板块的短期影响（客观，不夸大）

## 4. 看好标的（3~5 只，非必买清单）

| 代码 | 名称 | 现价 | 涨跌幅 | 理由（资金+估值+新闻各一句） |
|------|------|------|--------|------------------------------|

每只 1 行「风险点」。

## 5. 风险

- 板块退潮、政策落空、估值过高、个股暴雷等 2~4 条

> 仅供参考，不构成投资建议。看好≠建议买入，请自行决策。
```

---

## 选股原则（A 股）

- 优先 **趋势 + 资金 + 热点** 同向，不单看涨幅榜
- 回避 ST、连续一字板无法介入、纯题材无业绩且估值极端
- 用「相对板块强弱」而非绝对预测
