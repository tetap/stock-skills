# 板块分析与选股模板（专业 · 有立场）

用于 `/stock 电池板块最近走势，给我几个看好的股票` 等 **板块 + 选股** 请求。

---

## 写作标准

- §1 **先给板块评级**：**强势可跟 / 分化选强 / 退潮回避**
- 选股表必须写 **排序理由 + 相对板块强弱**，不是涨幅榜复述
- 每只候选股给 **介入区间 + 失效位**，禁止只写「可关注」
- 情绪面必须交叉：快讯 + 板块资金 + 概念/行业排行

## 禁止

- 踢皮球、只列板块名
- 必涨必买、全仓
- 顾问流派章节
- 「推荐 3 只」但无理由、无价位

---

## 必拉工具（≥12 次，再对候选股轻量扫描）

### 1. 定位板块

| 工具 | 参数 |
|------|------|
| `search_sectors` | query=用户说的板块词 |
| 若多个匹配 | 选最相关 1 个，报告中说明 |

### 2. 板块全景

| 工具 | 参数 |
|------|------|
| `get_sector_detail` | board_name, kline, limit 120 |
| `get_sector_detail` | board_name, fund_flow, limit 30 |
| `get_sector_detail` | board_name, members, limit 30 |
| `get_sector_overview` | concept + industry 各 limit 20 |

### 3. 新闻与情绪（必做）

| 工具 | 参数 |
|------|------|
| `get_market_news` | flash, limit 30, source=all |
| `get_market_news` | flash, keyword=板块词, limit 15 |
| `get_market_news` | xueqiu_hot, limit 10, source=xueqiu |
| `get_market_fund_flow` | limit 10 |
| `get_fund_flow_rank` | limit 20 |

### 4. 候选股轻量扫描（选 3~5 只）

从 **资金靠前 + 相对板块强势 + 非 ST** 里挑，每只至少：

| 工具 | 说明 |
|------|------|
| `resolve_symbol` + `get_realtime_quote` | 现价 |
| `get_valuation_metrics` | PE/PB |
| `get_stock_fund_flow` | 近 5 日主力 |
| `get_news_and_reports` | news, source=all, stock_name |
| 可选 | `get_kline` limit 60、`compare_performance` |

## 报告结构

```markdown
# {板块名} · 走势与选股

> 数据截止：{时间} · 板块评级：**{强势可跟|分化选强|退潮回避}** · 置信度：**{X}/10**

## 1. 结论

**一句话**：{板块为何强/弱，资金与新闻是否共振}

| 项目 | 内容 |
|------|------|
| 板块趋势 | 近 5/20/60 日（K 线） |
| 资金 | 板块主力 + 大盘背景 |
| 情绪 | 2~3 条快讯 + 雪球热榜是否含板块龙头 |
| 策略 | {跟龙头 / 挖低位补涨 / 不参与} |

## 2. 板块走势

| 近 5/20 日 | 涨跌家数 | 领涨股 | vs 大盘 |

## 3. 情绪与热点

- 与板块相关的 3~5 条快讯（带影响判断）
- 主题持续性：{一日游|可延续 1~2 周|中期逻辑}

## 4. 首选标的（3~5 只，按优先级排序）

| 优先级 | 代码 | 名称 | 现价 | 相对板块 | 核心理由 | 理想买点 | 失效位 |
|--------|------|------|------|----------|----------|----------|--------|

每只 1 句 **最大风险**。

## 5. 反面论证

- 板块逻辑不成立的 2 条理由
- 若板块指数跌破 XX，整份清单作废

> 仅供参考，不构成投资建议。
```

---

## 选股原则（A 股）

- **趋势 + 资金 + 热点** 三共振优先于纯涨幅
- 回避 ST、连续一字、无业绩纯题材且 PB 极端
- 龙头给「跟趋势」，补涨给「严格止损」
