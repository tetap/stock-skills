# A 股分析报告模板（专业 · 有立场）

面向 **A 股市场**的综合分析。流程：**拉数 → 多重审核**（[review-protocol.md](review-protocol.md)）→ **终稿 + §7 审核纪要**。

---

## 写作标准（必读）

### 要专业

- **开门见山**：§1 第一句必须是立场，不是背景介绍。
- **数据锚定**：每个判断后面跟数字（PE、主力净流入、MA 价位、涨跌幅），禁止空泛形容词。
- **交叉验证**：用「基本面 / 技术面 / 资金面 / 情绪面」四维度各给 **1 句带数据的结论**，再汇总。
- **相对视角**：必须写相对沪深300、相对所属板块的强弱（用 `compare_performance` + 板块涨幅）。
- **催化剂日历**：列出未来 2~4 个可验证节点（财报、股东大会、产品节点、政策窗口）。

### 要大胆（在合规内）

- **禁止**空泛踢皮球：「观望」「有待观察」「建议关注」单独出现而不给价位/条件 = 不合格。
- **禁止**必涨必买、全仓、稳赚。
- **允许且鼓励**明确立场用语：

| 评级 | 含义 | 适用场景 |
|------|------|----------|
| **积极跟踪** | 逻辑成立，可等点位或分批布局 | 质价趋势至少两维同向 |
| **右侧等待** | 逻辑 OK 但趋势/资金未确认 | 缺放量、缺均线收复 |
| **明确回避** | 不建议新建仓，持有者应减仓 | 估值极端+趋势破+资金流出 |
| **事件驱动博弈** | 仅适合短线，中线逻辑不成立 | 纯题材、亏损高 PB |

- **置信度**：§1 写 **终稿置信度 X/10**（初稿不得 >6）；须过 review-protocol 门禁。

---

## 禁止

- 踢皮球（空泛「观望」、redirect、只贴 JSON）
- 投资顾问流派章节
- 「必涨 / 必买 / 稳赚 / 全仓」
- 各节只写「尚可」「一般」而无数据

## 报告结构

**篇幅**：6 节完整；§1 可略长（核心），其余节 **表格填数 + 2~4 句解读**，禁止大段粘贴工具原文。

## 1. 投资结论（先读本节）

**一句话**：{20 字内立场，含最关键的矛盾点，如「增收不增利+空头排列，中线回避」}

## 内部轮次（给用户前必做，详见 [review-protocol.md](review-protocol.md)）

## §1 示例（国轩高科 · 风格参考，非实时报价）

> **一句话**：增收但 Q1 利润塌陷、588× PE 与空头趋势叠加 — **中线明确回避，仅 26 元平台可事件性博弈**。
> **评级**：明确回避 · **置信度**：7/10
> **理想买点**：26.0~26.5（近 20 日低点区） · **失效**：跌破 26.0 · **若错了**：Q2 扣非净利环比转正 + 放量站稳 MA20（≈29）
